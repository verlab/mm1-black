## MM1-BLACK v0.7.2

Same firmware as **v0.7.1** / **v0.7.0**. Web installer connect and flash reliability for CYD boards (CH340).

### Fixed

- **Web installer connect** (browser only; PlatformIO upload unchanged): **115200 baud** for CH340; DTR/RTS toggled separately (Chromium Linux bug); HW reset before esptool; timed connect retries; manual **BOOT + RST** path.
- **Linux Chrome**: banner with `google-chrome --disable-features=SerialSplitDtrAndRts` if auto-reset hangs.

### Install

1. Hard-refresh [Install MM1-BLACK](https://verlab.github.io/mm1-black/) (Ctrl+F5).
2. Close **PlatformIO upload/monitor** (free `/dev/ttyUSB0`).
3. Flash @ **115200 baud** → Install → CH340 port.
4. On prompt: **hold BOOT → tap RST → release RST → release BOOT**.
5. Linux Chrome still stuck: reopen browser with `--disable-features=SerialSplitDtrAndRts`.
