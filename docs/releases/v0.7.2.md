## MM1-BLACK v0.7.2

Same firmware as **v0.7.1** / **v0.7.0**. Web installer connect and flash reliability for CYD boards (CH340).

### Fixed

- **Web installer connect** (browser only; PlatformIO upload unchanged): **115200 baud** for CH340; DTR/RTS toggled separately; HW reset before esptool; timed connect retries; manual **BOOT + RST** path.

### Install

1. Hard-refresh [Install MM1-BLACK](https://verlab.github.io/mm1-black/) (Ctrl+F5).
2. Close **PlatformIO upload/monitor** (free `/dev/ttyUSB0`).
3. Flash @ **115200 baud** → Install → CH340 port.
4. On prompt: **hold BOOT → tap RST → release RST → release BOOT**.

**Assets:** `MM1-BLACK-denky32-v0.7.2.bin` + `.sha256`
