## MM1-BLACK v0.7.2

Same firmware as **v0.7.1** / **v0.7.0**. Web installer connect and flash reliability for CYD boards (CH340).

### Fixed

- **Web installer connect**: flash @ **115200 baud** for CH340 — Web Serial cannot change baud mid-session; 921600 caused ROM sync / connect failures.
- **Retries**: auto-reset sequence, then manual **BOOT + RST** prompt.
- **USB port filters** for CH340 / CP210x / FTDI; clearer on-page install steps.

### Install

1. Hard-refresh [Install MM1-BLACK](https://verlab.github.io/mm1-black/) (Ctrl+F5).
2. Keep flash speed at **115200 baud (recommended)**.
3. Close Serial Monitor / PlatformIO before flashing.
4. If connect fails: hold **BOOT**, tap **RST**, release RST, release BOOT, retry.

**Assets:** `MM1-BLACK-denky32-v0.7.2.bin` + `.sha256`
