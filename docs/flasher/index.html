Install MM1-BLACK

MM1-BLACK · ESP32 · USB · Web Serial

Firmware from GitHub Releases

Your browser does not support Web Serial. Open this page in Google Chrome or Microsoft Edge on a desktop PC (not on iOS).
  1. Connect MM1-BLACK (ESP32) with a USB data cable.
  2. Close any program that uses the device USB serial port (IDE, terminal, monitor).
  3. Pick flash speed below — 115200 is safest in the browser; IDE upload can use higher speeds.
  4. Click Install and select the correct USB serial port.
  5. When prompted, enter download mode: hold BOOT → tap RST → release RST → release BOOT.

USB on CYD boards is usually CH340 (WCH). The browser cannot change baud mid-flash like pio run -t upload (921600); it uses one speed for the whole session.
