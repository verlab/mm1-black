Install MM1-BLACK

MM1-BLACK · USB · Web Serial

Firmware from GitHub Releases

Your browser does not support Web Serial. Open this page in Google Chrome or Microsoft Edge on a desktop PC (not on iOS).
  1. Connect MM1-BLACK with a USB data cable.
  2. Close PlatformIO upload/monitor and any app using /dev/ttyUSB0.
  3. Keep flash speed at 115200 baud (required for CH340 in the browser).
  4. Click Install → pick the CH340 port (1a86:7523).
  5. When prompted, enter download mode: hold BOOT → tap RST → release RST → release BOOT.
