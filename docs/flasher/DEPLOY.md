# Deploying the flasher page

1. Merge PR with `.github/workflows/pages.yml` into `main`.
2. In the repo: **Settings → Pages → Build and deployment → Source: GitHub Actions**.
3. After the **Deploy flasher (GitHub Pages)** workflow succeeds, open:

   **https://verlab.github.io/cyd_brics5_mm1/**

If you see 404, Pages is not enabled or the workflow has not run yet.
